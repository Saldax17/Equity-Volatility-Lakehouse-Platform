# Config Folder

This folder contains configuration files for credentials and local paths.

## How to use this folder

1. Copy `config_template.py` and rename the copy to **config.py**
2. Edit `config.py` and insert your real credentials:
   - Alpaca API Key
   - Alpaca Secret Key
   - EODHD Token
3. DO NOT upload `config.py`.
   It is ignored thanks to `.gitignore`.

## Files
- `config_template.py` → safe for GitHub
- `config.py` → private, local-only
